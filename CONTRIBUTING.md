# Contributing to Simple Interest Calculator

Thank you for your interest in contributing to this project! We welcome all contributions that improve the project.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your changes.
3. Make your changes and test them.
4. Submit a pull request for review.

## Code of Conduct
Please follow the [Code of Conduct](CODE_OF_CONDUCT.md) when interacting with this project.

## Issues
If you find a bug or have a feature request, please open an issue in the repository.
